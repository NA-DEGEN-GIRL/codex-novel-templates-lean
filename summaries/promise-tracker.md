# 약속/계획 추적 대장

> 캐릭터 간 약속, 계획, 예정된 만남, 선언 등을 중앙에서 추적한다.
> 매 화 집필 후 갱신하며, 집필 전에 반드시 참조한다.

### 우선순위 기준

| 우선순위 | 의미 |
|---------|------|
| **높음** | 다음 1~3화 내 반드시 다뤄야 함 (독자가 기억하고 있음) |
| **중간** | 10화 이내에 진전이 필요함 |
| **낮음** | 장기 복선 성격. 아크 단위로 관리 |

## 갱신 규칙

### 추가 시점
아래 상황이 본문에 등장하면 **반드시** 새 항목을 추가한다:
- 캐릭터가 다른 캐릭터에게 약속/제안/맹세를 함
- 특정 시간/장소에서 만나기로 함
- 행동 계획을 세움 ("내일 ~하자", "보름 후 ~에 가겠다")
- 조건부 약속 ("~하면 ~해주겠다")
- 선언/결심 ("반드시 ~하겠다") — 다른 캐릭터가 들은 경우

### 갱신 시점
매 화 집필 후 아래를 확인한다:
1. 이번 화에서 **새 약속**이 생겼는가? → 활성 약속에 추가
2. 기존 약속에 **진전**이 있었는가? → 상세 컬럼에 진전 내용 추가
3. 약속이 **이행/무효화**되었는가? → 완료된 약속으로 이동
4. 약속 **시한이 다가오는** 항목이 있는가? → 우선순위 상향

### 검증 시점
- **unified-reviewer (continuity 모드)**: 활성 약속 중 예정 회수 화수를 넘긴 항목이 없는지 검증
- **집필 전**: 우선순위 "높음" 약속이 이번 화에서 다뤄지는지 확인
