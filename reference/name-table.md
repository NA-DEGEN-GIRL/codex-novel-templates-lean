> AI 소설 기획 시 이름 선택 참고 테이블. INIT-PROMPT에서 캐릭터/장소/조직 설계 시 활용.
> 최종 확정된 이름은 settings/03-characters.md, settings/04-worldbuilding.md에 반영.

## How to Use
- 용도(인명/지명/AI명 등) → 문화권 → 시대/성별 순으로 탐색
- 태그로 뉘앙스 필터링
- 이름을 그대로 쓰거나 변형하여 사용

## 0. 작명 원칙 (AI 함정 회피)

> AI가 이름을 만들 때 빠지는 5가지 함정과 대응법.

### 함정 1: 단어 직접 조합
❌ Shadow + Blade = Shadowblade, 검(劍) + 혼(魂) = 검혼
✅ 의미보다 **소리의 질감과 리듬**이 캐릭터와 맞는지 확인. 뜻이 없어도 소리가 주는 느낌이 중요 (사우론, 볼드모트).

### 함정 2: 발음의 온도 무시
- 강한 캐릭터: 파열음 (ㄱ, ㅋ, ㅌ, ㅍ) — 가츠, 게롤트, 크툴루
- 신비/우아: 유음·비음 (ㄴ, ㄹ, ㅁ, ㅇ) — 갈라드리엘, 에어리스, 렘
- 입이 크게 벌어지는 발음 = 개방감 / 닫히는 발음 = 폐쇄적·비밀스러움

### 함정 3: 문화권 혼용
한 소설 안에서 이름의 어원이 중구난방이면 세계관이 깨진다. 지역별로 어원 계열을 통일할 것 (A지역=북유럽풍, B지역=라틴어 계열 등).

### 함정 4: 완전한 가상어만 사용
실제 이름이나 단어를 살짝 비튼 이름이 더 기억에 남음. John → Jon, Alice → Aerith. 익숙하면서도 어딘가 다른 느낌이 호기심을 자극.

### 함정 5: 별명/약칭 미고려
긴 이름을 누가 어떻게 줄여 부르는지가 관계를 보여줌. 알렉산드로스를 '알렉스'로 부르는 친구와 '폐하'로 부르는 신하가 다르다.

## 0.7 이름 조합 규칙

> 이름 후보만 많으면 AI가 문화권 혼합, 어색한 성명 순서, 과한 별칭을 만든다.

### 성+이름 순서
| 문화권 | 순서 | 예시 |
|--------|------|------|
| 한국/중국/일본 | 성 + 이름 | 진무원, 李墨寒, 키류 카즈마 |
| 서양/러시아 | 이름 + 성 | Geralt of Rivia, 드미트리 볼코프 |
| 무협 별호 | 별호 + 이름 or 별호만 | 천검 무현, 혈귀 한소 |

### 문화권 혼합 금지
- ❌ 일본식 이름 + 중국식 성 (사쿠라 李)
- ❌ 한국식 이름 + 서양식 성 (서경 Blackwood)
- ❌ 같은 세계관 안에서 문화권이 뒤섞인 이름들
- ✅ 혼혈/이중국적 설정이 있으면 의도적 혼합 가능

### 호/별칭/칭호 체계
| 유형 | 위치 | 예시 | 언제 사용 |
|------|------|------|----------|
| 호(號) | 이름 대신 | 천검(天劍) | 무협 — 무공/특징 기반 |
| 별호 | 이름 앞 | 독야(獨夜) 무현 | 무협 — 명성 획득 후 |
| 법명/도호 | 이름 대신 | 현각(玄覺) | 종교/수행자 |
| 콜사인 | 이름 대신 | Cipher, Ghost | SF/군사 |
| 코드네임 | 이름 대신 | Agent-7, 2B | SF/스파이 |
| 작위/칭호 | 이름 앞 | Sir Gideon, 왕자 이산 | 판타지/사극 |

### 애칭/줄임말 생성
- 3음절 이상 → 2음절로: 알렉산더→알렉스, 아나스타시아→나스탸, 세바스찬→세바
- 친밀도에 따라 호칭 변화 = 관계 묘사 장치
- AI에게: "이 캐릭터를 부르는 방식이 관계마다 다르게" 지시하면 효과적

### 같은 그룹 내 이름 세트 규칙
- 형제/가문: 공통 음절 또는 돌림자 (예: 현우/현서/현진)
- 조직원: 코드네임 패턴 통일 (예: 색깔 계열, 별자리 계열)
- 적과 아군: 음감으로 구분 (아군=부드러운 음, 적=파열음)

## 0.8 장르별 어감 가이드

> 장르마다 "좋은 이름"의 기준이 다르다. AI가 기본값으로 만드는 이름은 대부분 판타지풍이므로, 다른 장르에서는 의식적으로 조정해야 한다.

### 무협
- **음절**: 2~3음절. 4음절 이상은 별호에만.
- **자음**: 선명하고 강한 초성 (ㄱ, ㅈ, ㅎ, ㅊ)
- **한자**: 의미가 캐릭터를 암시하되 직역하지 않음. "검혼"보다 "무현"이 나음.
- **호/별호**: 무공이나 특징 기반. 초반에는 없고 명성을 얻으면 붙음.
- **금지**: 현대 한국식 이름 (민준, 서연), 영어 외래어

### 정통 판타지
- **음절**: 2~4음절. 장중한 이름일수록 길게.
- **자음군**: 서양 판타지는 자음 클러스터 (Galadriel, Thranduil)
- **가문명**: 중요. 이름보다 성이 세계관을 보여줌.
- **종족별 음감**: 엘프=유음, 드워프=파열음, 오크=탁음
- **금지**: 너무 현실적인 이름 (John the Warrior)

### SF/사이버펑크
- **음절**: 짧고 식별성 높게. 2음절 선호.
- **코드명/모델명**: 병행 사용. 본명은 나중에 공개.
- **숫자/약어**: 허용. 2B, K-9, TARS.
- **기업명**: 깔끔하고 차가운 인상. Nexus, Axiom, Helion.
- **금지**: 과도한 판타지풍 이름

### 현대물/로맨스
- **음절**: 실제 한국인 이름 느낌. 과하지 않게.
- **세대감**: 20대=현대적, 40대=약간 고전, 60대=확실히 옛날
- **희귀도**: 주인공은 살짝 독특, 조연은 평범. 전부 희귀하면 이상함.
- **금지**: 무협식 한자 이름, 이세계풍 이름

### 호러/다크
- **음감**: 마찰음(ㅅ, ㅆ, ㅎ), 어두운 모음(ㅓ, ㅡ, ㅜ)
- **의미 중첩**: 이름 자체에 복선 가능 (예: 멸각=소멸의 저주)
- **금지**: 밝고 경쾌한 이름 (단, 대비 효과로 의도적 사용은 가능)
