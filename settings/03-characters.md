# 캐릭터 마스터시트

> 모든 등장인물의 설정을 관리한다. 새 캐릭터 등장 시 여기에 추가한다.
> 대표 대사와 말투 메모는 복붙용 사전이 아니다. 어휘 선택, 높낮이, 위계감, 숨 고르는 위치를 확인하기 위한 **운용 앵커**로 사용한다.
> "짧고 건조하다", "다정하다" 같은 총평만으로 끝내지 말고, **부모/연장자/동년배/아랫사람/적대자** 앞에서 어떻게 달라지는지까지 적는다. `CODEX.md`의 `§8.1 호칭/어투 매트릭스`와 서로 맞아야 한다.

## 캐릭터 시트 형식

각 캐릭터는 아래 형식으로 작성한다.

```markdown
### [캐릭터명]

- **역할**: 주인공 / 히로인 / 조력자 / 적대자 / 조연
- **나이**:
- **외모**: (한줄 요약)
- **성격**: (핵심 특성 2~4가지)
- **말투**: (구체적 특징. 존댓말/반말, 특이한 어미, 말버릇 등)
- **말 길이 경향**: (짧게 자르는지, 설명이 길어지는지, 감정 시 길이 변화가 있는지)
- **동기**: (이 캐릭터가 원하는 것)
- **약점**: (내면적/외면적 약점)
- **금기/트리거**: (이 캐릭터가 절대 하지 않는 것, 또는 특정 반응을 유발하는 것)
- **회피 반응**: (곤란한 질문/압박을 받을 때 어떻게 피하는지)
- **관계별 말투 규칙**:
  - 부모/가족:
  - 연장자/사부/상급자:
  - 동년배/친한 사이:
  - 아랫사람/후배:
  - 적대자/대치 상황:
- **대화 대비축**: (상대나 상황이 바뀔 때 말투/리듬/압력이 어떻게 달라지는지)
- **타인이 보는 인상**: (다른 캐릭터들이 이 인물을 처음 봤을 때 받는 인상)
- **비밀**: (다른 캐릭터가 모르는 것)
- **관계**:
  - [캐릭터A]: 관계 설명
  - [캐릭터B]: 관계 설명
- **변화 아크**: (작품 전체에서의 성장/변화 방향)
- **대표 대사 2~3종**: (보이스 드리프트 방지용 — 이 캐릭터의 어투를 가장 잘 보여주는 대사)
  1. "{{대사1}}" — {{상황 설명}}
  2. "{{대사2}}" — {{상황 설명}}
  3. "{{대사3}}" — {{상황 설명}}
```

## 캐릭터 규칙

1. **말투 일관성**: 각 캐릭터의 말투는 감정 변화에 따라 약간 변할 수 있지만 기본 패턴은 항상 유지한다. **집필 전 대표 대사 3종 + 관계별 말투 규칙을 함께 확인**하여 어투를 샘플링한다.
   - 대표 대사를 그대로 반복하지 않는다. 같은 인물이라도 장면에 맞는 변주를 새로 만든다.
   - "짧고 건조하다" 같은 기본 톤은 출발점일 뿐이다. 부모, 연장자, 사부, 아랫사람, 적대자 앞 register까지 적혀 있어야 실제 장면에서 덜 샌다.
2. **성격 일관성**: 갑작스러운 성격 변화는 금지. 변화는 사건을 통한 점진적 과정이어야 한다.
3. **능력 일관성**: 캐릭터의 능력은 기존 설정과 장면 맥락 안에서 발휘된다. 예외적 상승이나 붕괴는 원인과 대가가 서사에 제시되어야 한다. 갑작스러운 파워업은 사전 복선 필수.
4. **죽은 캐릭터**: 사망 처리된 캐릭터는 원칙적으로 재등장 불가. 단, 세계관 규칙과 사전 단서가 있는 경우(부활, 회귀, 빙의, 환영, 꿈, 복제, 영혼 등) 허용. 회상/언급은 항상 가능.
5. **신규 캐릭터 등록**: 이름이 있는 캐릭터가 새로 등장하면 반드시 이 시트에 추가한다.
